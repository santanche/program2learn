How to read these solutions

Each page shows the running simulation with setupCorrelations() fully implemented and displayed in a code panel at the bottom. The function is the only place where component relationships are defined — everything else is rendering infrastructure.

Compare the three versions to see how the same correlations are expressed differently across paradigms.

EVENT-BASED

HTMX

Correlations defined as a chain of event listeners using on() and emit().

 STATE-BASED

Vue

Correlations expressed as computed() derivations and a watch() for side effects.

 STREAM-BASED

RxJS

Correlations built as a stream pipeline — each component is an observable derived from the previous.
