About this Lab

This lab simulates a Supervisory Control and Data Acquisition (SCADA) industrial system with a machine, thermometer, cooler, and alert light. The same simulation is implemented three times — each version uses a different reactive programming paradigm.

  • A switch turns the machine on or off.
  • When the machine is on, temperature rises every 10 seconds: low → medium → high.
  • When temperature reaches high, the cooler activates automatically.
  • The alert light reflects the current temperature level.

In each version, find the setupCorrelations() function — it is the core of the reactive logic, and the only place where component relationships are defined.

EVENT-BASED

HTMX

Custom DOM Events

Components communicate by emitting and listening to named events. Changes propagate as discrete event dispatches through the document.

 STATE-BASED

Vue

Reactive State & Computed

A shared reactive state drives all visual updates. Watchers and computed properties automatically keep everything in sync.

 STREAM-BASED

RxJS

Observable Streams

All inputs and outputs are observable streams. Operators transform and combine them to produce derived behaviors.
