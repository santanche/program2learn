About this Lab

This lab simulates a Supervisory Control and Data Acquisition (SCADA) industrial system — a machine, thermometer, cooler, and alert light. The same simulation is implemented three times, each with a different reactive programming paradigm:

  • HTMX — event-based: components communicate through custom DOM events
  • Vue — state-based: a reactive state drives all visual updates via computed properties and watchers
  • RxJS — stream-based: all behaviors are observable streams composed with operators

In every version, setupCorrelations() is the single function that defines how components relate to each other.

✎ Exercise

Challenge

The simulation runs but nothing reacts. Write setupCorrelations() in the browser editor, click Run, and test your implementation by clicking the switch.
