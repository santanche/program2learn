▸ AVAILABLE API (click to expand)

RxJS creation operators:

fromEvent(element, 'click') — stream of click events
timer(delay, period) — emits 0, 1, 2, … starting after delay, then every period ms
of(...values) — finite stream of fixed values, then completes

Pipeable operators (use inside .pipe(...)):

scan((acc, val) => newAcc, seed) — accumulate state across emissions
switchMap(val => observable$) — switch to a new inner stream (cancels previous)
map(val => transformed) — transform each value
take(n) — complete after n emissions
share() — multicast so multiple subscribers share one execution

Subscription (use instead of .subscribe()):

sub(observable$, handler) — tracked subscribe; all are cancelled on Reset
