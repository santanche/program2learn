▸ AVAILABLE API (click to expand)

Functions you can call:

on('eventName', handler) — subscribe to a custom event
emit('eventName', detail) — dispatch a custom event with a detail object
startTemperatureRise() — start the 10-second temperature rise timer
resetTemperature() — stop the timer and reset temperature to 'low'

Event names and detail shapes:

'switch:clicked' — { } (fired when switch image is clicked)
'machine:changed' — { on: boolean } (emit this to update machine visuals)
'temperature:changed' — { level: string } (fired by startTemperatureRise)
'cooler:changed' — { on: boolean } (emit this to update cooler visuals)
'alert:changed' — { level: string } (emit this to update alert light)

Level values:

LEVELS = ['low', 'medium', 'high']
