▸ AVAILABLE API (click to expand)

Reactive primitives (available as globals):

watch(ref, handler, options?) — run handler whenever ref changes
watch(ref, handler, { immediate: true }) — also run handler immediately on setup

Reactive state passed as arguments:

switchOn ref<boolean> — true when switch is in ON position (toggled by clicking)
machineOn ref<boolean> — you must keep this in sync with switchOn
tempLevel ref<string> — 'low' | 'medium' | 'high' (managed by the temperature helpers)
coolerOn ref<boolean> — you must keep this in sync with tempLevel

Helper functions:

startTemperatureRise() — sets tempLevel to 'low' and starts 10-second timer
resetTemperature() — stops timer, resets tempLevel to 'low'
